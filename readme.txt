가계도 개발

개발환경:웹앱,핸드폰도 사용,자바스크립트,리액트

- 나를 기본으로 하고 밑에 탭을 만들어 여러개 만들 수 있도록
- 색상 : 남자 파란색, 여자 빨간색
- 위치 남편은 좌측으로, 부인은 우측으로 배우자 추가시 자동으로
- 부모는 위쪽으로 자녀는 아래쪽으로
- 부인과 남편은 재혼할 수도 있으므로 여러명 추가 가능토록
- 모든 사람을 선으로 연결
- 형제는 옆으로 만들되 부모에서 선 두개가 내려와 하나로 합쳐쳐 각개 자녀로 분할되도록
- 사람마다 각개 ID 부과하여 관리할 수 있도록, 남자는 M으로 시작, 여자는 W로 시작 , M생년월일-001 이런식으로 M20240620-001
